## Daily plan (HomeBot Pi dashboard)

Today's todos live in `memory/YYYY-MM-DD.md` under a `## Plan` section.

- Use checkbox lines: `- [ ]` pending, `- [x]` done
- Format, tokens, carry-forward: **read `skills/daily-plan/SKILL.md` when editing the plan**
- Do not store daily todos only in `MEMORY.md`
- Pi dashboard: `http://127.0.0.1:8080`

## Skills — load on demand only

Read skill files when the request matches. Do not summarize all skills each turn.

| Skill | Use when |
|-------|----------|
| `daily-plan` | todos, plan changes |
| `homebot-events` | `## Events` reminders |
| `homebot-media` | image archive/purge |
| `homebot-setup` | install/fix HomeBot |
