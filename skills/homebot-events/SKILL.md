# HomeBot One-off Events (optional)

Use `## Events` for **extra** timed reminders (appointments, meetings):

```markdown
## Events
- 14:00 DENTIST — Downtown clinic {remind:10,0}
```

Daily todos live in `## Plan` — see `skills/daily-plan/SKILL.md`.
